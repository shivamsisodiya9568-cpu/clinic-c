# GS Clinic Website

A premium, modern, fully responsive static website for **GS Clinic** in Agra, Uttar Pradesh, India.

## Features

- Animated hero section with premium medical visual design
- Sticky responsive navbar with mobile menu
- About clinic and doctor profile sections
- Services grid with modern medical cards
- Appointment booking form with JavaScript feedback
- Floating **GS Clinic AI Assistant** chatbot
- Google Map embed for Agra
- Patient reviews, gallery, health tips, and FAQ accordion
- WhatsApp button, call now button, and back-to-top button
- Loading animation, smooth scrolling, hover effects, glassmorphism, gradients, and responsive layouts

## File Structure

GS Clinic Website/

## How To Run

Open `index.html` in any modern browser.

No build tools, package installation, or server setup is required.

## Customization Guide

Update these placeholders before publishing:

- Phone and WhatsApp: replace `+91 00000 00000` and `910000000000`
- Email: replace `hello@gsclinic.example`
- Doctor name and qualification in the Doctor Profile section
- Clinic address if you want a more exact Agra location
- Gallery images, services, reviews, and health tips
- Google Map query in the iframe if you have an exact address

Main files to edit:

- `index.html`: content, sections, links, images, map, and contact details
- `style.css`: colors, spacing, layout, animations, and responsive design
- `script.js`: mobile menu, form behavior, FAQ accordion, chatbot answers, and scroll actions

## GitHub Pages Deployment

1. Create a new GitHub repository.
2. Upload `index.html`, `style.css`, `script.js`, and `README.md` to the repository root.
3. Open the repository on GitHub.
4. Go to **Settings**.
5. Open **Pages**.
6. Under **Build and deployment**, select **Deploy from a branch**.
7. Select the `main` branch and `/root` folder.
8. Save the settings.
9. GitHub will publish the website and provide a public URL.

## Notes

The contact number, WhatsApp number, email, and doctor details are placeholders. Replace them with real clinic information before production use.
